# Portfolio

A modern personal portfolio built with React, TypeScript, and Vite, ready for Azure Static Web Apps.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Azure deployment

This project includes a GitHub Actions workflow for Azure Static Web Apps in [.github/workflows/azure-static-web-apps.yml](.github/workflows/azure-static-web-apps.yml).

Set the `AZURE_STATIC_WEB_APPS_API_TOKEN` secret in GitHub, then push to `main` to deploy.

## Customize

- Replace `Your Name` with your name.
- Update the placeholder email, links, skills, projects, and experience.
- Add screenshots or project links if you want richer project cards.
